    • index_3

      Installation d'un package : canvas-sketch

      Pour lancer le serveur : canvas-sketch nomDuFichier.js si le fichier n'est pas créé, il suffit de rajouter --new (pour créer un nouveau fichier) et --open (pour l'ouvrir) en bout de commande

      On peut sauvegarder ce que l'on dessin avec la commande ctrl+s, un fichier.png est générer à partir du canvas

      Avec cette commande : canvas-sketch nomDuFichier.js --output=output/01 tout nos fichiers sauvegarder iront dans ce dossier. Cela rafraichi la page!

      Canvas sketch nous permet aussi de faire des impressions. Dans notre objet settings, il suffit de changer la valeur de notre propriété, au lieu d'un tableau, mettre "A4",d'ajouter de la densiter pour nos pixels avec pixelsPerInch:300 et pour l'orientation orientation:"landscape"

      Tout est disponible sur Github et W3schools
